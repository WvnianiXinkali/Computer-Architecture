# Prime Numbers in a Given Range

## Problem Statement  
You are given two integers **N** and **M**, where **\( 2 \leq N < M \leq 1000 \)**.  

Write a program that prints all **prime numbers** in the range **[N, M]**, including both boundaries.  

## Input Example  
25 45

## Output Example  
29 31 37 41 43
